Если поле deleted == false, то сообщение считается удаленным
Если поле deleted == true, то сообщение не удалено, оно существует
Id нумерация с нуля!
Реализация чата такова, что поле id не может быть опущенно. Т.к. если сообщение удаляется, то его 
"тело" существут, только в поле deleted == true. Т.е. нумерация берется относительно номера List!

Host: localhost:999
User-Agent: Fiddler

{"date":"null","deleted":"false","author":"null","id":"null","editDelete":"true","message":"hi"}

Host: localhost:999

{"id":"1"}
