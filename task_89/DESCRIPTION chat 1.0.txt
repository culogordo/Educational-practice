Многопользовательский вариант чата
Поддержка многострочного ввода
Отправка сообщения на ctrl+Enter
Дополнительно поле "ДАТА" (время измененного сообщения, принятого сообщения)
Удаляет тэги (не поддерживает передачу тэгов). Если сообщение содержит теги, то передастся только сообщение без тега
Проверка валидности имени пользователя (только латинские буквы длинной от 3 до 20 символов, цифры, _(подчеркивание)
Невозможность иных действий во время редактирования сообщения, кроме приема сообщений.
Гибкая верстка (responcive web design)
User Experience (подсвечивающиеся фрагменты, подсказки, фокусировка на отдельные элементы)
Невозможность отправки пустых сообщений (в том числе сообщений, состоящих из new line)

В случае отключения сервера (JavaChatSimulator) приложение становится неактивным, т.к. отсутствует альтернативная БД
